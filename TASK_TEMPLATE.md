# Task Template

Use this when starting a fresh Codex thread for this repo.

## Before You Start

1. Update `PLANS.md` `Current Objective` if needed.
2. Start a new Codex conversation.
3. Paste the prompt below.

## Kickoff Prompt

```text
Please read PROJECT_BRIEF.md, AGENTS.md, and PLANS.md first. Before changing code, read the relevant files in docs/system, and update docs/system in the same change for any architecture, module, API, data-flow, plugin, or infra change.
```

## Notes

- If `PLANS.md` and the actual task differ, update `Current Objective` first.
- Do not describe this repo as a planner/router/specialist agent system unless code has been added to make that true.
